# FOCES Technical Events Hub

A responsive technical events portal built for FOCES students to view and track upcoming hackathons, seminars, and workshops in one place.

## 🔗 Submission Links
- **Live Deployment:https://arya892.github.io/foces-events-portal/
- **Demo Video:https://drive.google.com/file/d/1pE51nHBHwFgpODPKL-PnutbSvwko3KCY/view?usp=sharing

## 🧠 Design & AI Tooling Choice
- **Frontend Stack:** Built using React and Tailwind CSS for rapid UI development and fluid mobile responsiveness.
- **AI Assistance:** **Claude** was utilized as a pair-programmer to assist in generating mock event data structures, optimizing the state-based search/filter logic, and ensuring clean Tailwind CSS breakpoints.

## 📱 Features
- Fully responsive layout (Desktop grid to Mobile single-column).
- Instant search by event name or keyword.
- Category filtering tabs (Hackathons, Workshops, Seminars).
- Interactive "Register" pathways.
